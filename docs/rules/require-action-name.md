> Require action name to be set.

## :book: Rule Details

This rule reports when action has no name.

::: correct

```yaml
name: CI

on:
  push:
    branches:
      - main
```

:::

::: incorrect

```yaml
# empty file
```

:::

::: incorrect

```yaml
# no name
on:
  push:
    branches:
      - main
```

:::

::: incorrect

```yaml
# name is empty
name:

on:
  push:
    branches:
      - main
```

:::

## :wrench: Options

Nothing.
